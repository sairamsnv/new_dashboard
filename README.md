# rare_wms_dashboard

Warehouse / dashboard monorepo: Django backend (`backend/`), legacy frontend (`frontend/`), and new Vite UI (`new_fronend/`).

## Quick start

- **Backend:** `cd backend` → activate venv → `python manage.py runserver`
- **New UI (dev):** `cd new_fronend` → `npm install` → copy `.env.example` to `.env` → `npm run dev` (or `npm run build` for production assets)
